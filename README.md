## 📌 Overview

This project simulates a **banking behavioural risk and ALM framework**, inspired by real-world financial risk practices used in institutions such as ING.

It focuses on:
- Customer behavioural modelling (prepayment risk)
- Asset & Liability Management (ALM)
- Interest rate risk (IRRBB)
- Survival analysis of customers
- Stress testing under economic scenarios

---

## 🧠 Key Components

### 1. Behavioural Risk Model
- Logistic regression for prepayment probability
- Customer-level behavioural drivers

### 2. ALM Simulation
- Deposit vs withdrawal modelling
- Balance sheet evolution

### 3. IRRBB Stress Testing
- Interest rate shock scenarios
- Cashflow sensitivity analysis

### 4. Survival Analysis
- Customer retention modelling
- Kaplan-Meier estimator
